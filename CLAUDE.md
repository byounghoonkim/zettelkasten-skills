# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

`zettelkasten-skills` is a collection of Claude skills for applying modern digital Zettelkasten methodology to local markdown note systems (Obsidian, plain `.md` files, etc.).

## Skill

The primary skill lives in `skills/zettelkasten/`:

| File | Purpose |
|------|---------|
| `SKILL.md` | Main workflow — capture mode and connect mode |
| `note-types.md` | Note type templates and quality criteria |
| `naming.md` | Frontmatter schema and filename conventions |

## Skills Development

When creating or editing skills in this repository, use the `superpowers:writing-skills` skill to ensure correct structure and deployment verification.
